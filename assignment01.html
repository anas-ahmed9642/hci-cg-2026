Purpose

The purpose of Lab & Assignment 01 is to bring all students to the same baseline level of semantic HTML skill with minimal CSS and no JavaScript. Your work should clearly communicate the differences between Norman’s model, GOMS, and KLM.

Starter Project

You will receive a starter website repository (5 pages). Your task is to improve the HTML structure, clarity, and accessibility while keeping within the rules below.

  • index.html
  • cv.html
  • contact.html
  • analysis.html
  • norman2klm.html

Accessibility Requirements

  • Use aria-current="page" on the active navigation link
  • All form controls must have associated <label>
  • Images must have appropriate alt text
  • The site must be keyboard usable

CSS Rules (Strict: Minimal Only)

You may use CSS only for:

  • color
  • background-color
  • font-family
  • font-size

Not allowed:

  • No margin, padding, border, flex, grid, etc.
  • No animations or transitions
  • No frameworks
  • No JavaScript

Submission

  • Submit GitHub repository link
  • Must render correctly via GitHub Pages
  • Due: Week 05
